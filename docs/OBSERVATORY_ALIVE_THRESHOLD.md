# Cuándo el Observatory empieza a sentirse vivo

El punto donde SlipUp **empieza a sentirse “vivo”** no depende de millones de momentos. Ocurre sorprendentemente pronto: cuando la ventana de 48 h tiene **entre ~300 y ~500 momentos compartidos**.

## Fases por masa (48 h)

| Ventana (48 h) | Sensación | Qué ocurre |
|----------------|-----------|-------------|
| **&lt; ~100** | Cielo vacío | El grado salta por evento; Nearby/Horizon casi vacíos; mezcla type×mood inestable. Se percibe como *instrumento experimental*, no como campo humano. |
| **~100 – 300** | Textura | Aparecen mezcla, pequeños patrones, señales repetidas. Nearby muestra cosas como *Working alone*, *Going to the gym*. Sensación de *campo incipiente*, aún no del todo real. |
| **~300 – 500** | **El sistema cobra vida** | Suficientes combinaciones type×mood, repetición natural, masa para inercia. Nearby casi siempre con momentos; Horizon con lecturas consistentes. El usuario percibe: *“esto está pasando de verdad”*. El cielo deja de sentirse simulado. |
| **~1000+** | Pleno funcionamiento | Grado estable, Nearby con señal, Horizon claro, Strata con sentido. El campo se sostiene solo; no depende de usuarios individuales. |
| **10k – 100k** | Meteorología | El clima se vuelve *pesado*: movimientos del grado pequeños pero significativos. Se siente más como meteorología y menos como app. |

## Por qué ~400 momentos es el umbral

Con ~400 momentos en 48 h:

```
400 / 48 h ≈ 8 por hora
```

- Siempre hay actividad reciente.
- La mezcla de señales es estable.
- El clima tiene masa.

La atmósfera empieza a parecer **continua**.

## Crecimiento orgánico

Muchos sistemas necesitan *100k usuarios activos* para funcionar. SlipUp puede sentirse real con **~200–300 personas contribuyendo** si cada una aporta algunos momentos al día.

## Qué ayuda a llegar a la masa inicial

1. **Panel Contribute claro**
2. **Nearby mostrando humanidad real**
3. **Clima con inercia** (masa suficiente para que el grado no salte por cada evento)

Si esas tres piezas funcionan, el campo empieza a sostenerse.

## Señal de que SlipUp ya está vivo

Cuando aparecen momentos como *Working alone*, *Running late*, *Too much coffee*, *Need fresh air* **sin que nadie los haya puesto manualmente**, el sistema deja de parecer producto y empieza a parecer **atmósfera humana**.
